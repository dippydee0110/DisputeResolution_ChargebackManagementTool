🚀 Quick Start Guide

Step 1: Generate Excel Dataset

Run this command to generate sample chargeback records:

python scripts/generate_chargeback_dataset.py

Output: output/chargeback_records.xlsx

Step 2: Add Customer Chat Sessions

Place your chatbot transcripts in evidence folders:

evidence/ CASE123/ chat_CASE123.txt CASE124/ chat_CASE124.txt ...

The system scans for chat_*.txt files automatically.

Step 3: Run Early Dispute Detection

Analyze sessions and generate predictions:

python scripts/early_dispute_detection.py

Output: output/early_dispute_detection.html

Step 4: Run Complete Analysis (All Steps)

Or run everything at once:

python scripts/run_analysis.py

Generates all reports in one command.

Step 5: Review Reports

Open these files in your browser and Excel:

  • 📊 chargeback_records.xlsx — Your chargeback data with risk scores
  • 🔍 early_dispute_detection.html — Risk analysis from chat sessions
  • 📋 dispute_detection_results.json — Machine-readable results

⚠️ Detected Signals (AI Looks For These)

The system analyzes chat sessions and flags these early warning indicators:

🔴 Customer doesn't recognize transaction
🔴 Refund request (before chargeback)
🔴 Item not received
🟠 Duplicate charge
🟠 Quality complaint
🟠 Merchant name mismatch
🟠 Frustration escalation
🟠 Urgent/emergency tone

🎯 Using Your Own Data

  1. Replace the sample data in output/chargeback_records.xlsx with your actual chargeback records
  2. Add customer chat transcripts to evidence/CASEID/chat_CASEID.txt folders
  3. Run: python scripts/early_dispute_detection.py
  4. Review the generated output/early_dispute_detection.html in your browser
  5. Cases are scored by risk level - prioritize the 🔴 CRITICAL ones for immediate action
