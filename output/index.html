How to Use This System

For Administration:

  • Start with Merchant Admin Dashboard for overview
  • View case statistics and risk distribution
  • Click case ID to view specific customer page
  • Export data for reporting and analysis

For Customer Communication:

  • Select appropriate risk-level template
  • Customize with customer-specific details
  • Share via email, support portal, or SMS
  • Track customer actions and responses

For Analysis:

  • Review Early Dispute Detection report
  • Identify early warning signals
  • Take proactive measures before chargebacks
  • Monitor trends over time
